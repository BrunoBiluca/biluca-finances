# RF 02.04 - Checagem de duplicatas

__Descrição__
Como usuário quero que o sistema verifique entradas duplicadas sempre que eu entrar com importar algum extrato.

__Impacto__
Muitas vezes os extratos podem repetir registros, apresentar ao usuário esses registros duplicados permite que a base fique mais limpa e consistente.
