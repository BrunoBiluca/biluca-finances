# RF 01.03.01 - Tipos de identificações

__Descrição__
Como usuário quero poder configurar o tipo que a identificação define.

Identificações podem ser:

- Receitas
	- Todos os registros de identificações de receitas tem o valor como positivo
- Despesas
	- Todos os registros de identificações de receitas tem o valor como negativos
- Investimentos
	- Os investimentos tem valores negativos já que eles são uma movimentação de uma conta para outra, porém eles são ignorados na hora calcular os relatório já que eles não representam uma receita
