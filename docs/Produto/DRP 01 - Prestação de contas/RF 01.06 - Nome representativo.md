# RF 01.06 - Nome representativo

__Descrição__
Como usuário quero poder alterar o nome de qualquer registro de contabilidade.

Nesse caso, precisamos manter o nome proveniente do extrato e criar um novo campo para armazenar o nome representativo.

__Impacto__
Dessa forma o usuário pode mantem a [[RF 02.04 - Checagem de duplicatas]] enquanto tempo registros mais bem definidos ajudando nas análises.
