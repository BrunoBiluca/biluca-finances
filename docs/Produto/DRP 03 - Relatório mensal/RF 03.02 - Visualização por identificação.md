# RF 03.02 - Visualização por identificação

__Descrição__
Apresentar as informações de contabilidade por identificação.

Ordenados da identificação mais relevante até o menos relevante.

__Impacto__
Discretizar cada identificador permite ao usuário verificar no detalhe cada identificador.

Isso permite verificar alterações no padrão por identificador, identificando discrepâncias e formulando alterações no comportamento.
